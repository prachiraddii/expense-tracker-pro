# 💰 AI Expense Tracker Pro

Welcome to my JavaScript-based project! This application helps users manage their daily expenses efficiently with smart insights, charts, and secure login functionality.

---

## 📌 About

This project is built using HTML, CSS, and JavaScript to provide a complete expense management solution. It allows users to track spending, visualize data, and receive intelligent insights about their financial habits.

---

## 🔗 Live Demo

https://prachiraddii.github.io/expense-tracker-pro/

---

## 📁 Features

* 🔐 User Login Authentication
* 💸 Add, delete, and categorize expenses
* 📊 Interactive Pie Chart (Chart.js)
* 🤖 AI-based spending insights
* 🌙 Dark Mode toggle
* 💾 Data stored using LocalStorage
* 📱 Simple and responsive UI

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Chart.js
